Help & Feedback

In certain cases, the extension may encounter some operational issues. If you experiencing any issues, check the steps below:

1. Open the Chrome extension page ("chrome://extensions/" in the address bar)

2. Find "Awesome Screenshot"

3. Disable Awesome Screenshot, then re-enable it

4. Refresh the page you want to capture with Awesome Screenshot, and try to take a screenshot again

5. Make sure that flash is enabled for your browser.

If you experience problems, or if you have other suggestions, don't hesitate to shoot us an email at care@awesomescreenshot.com or on Twitter @awe_screenshot. If you are reporting an issue, include details about how we might reproduce the issue you experienced. If you would like additional features, describe a usage scenario for a specific feature.
